プライバシーポリシー

最終更新日: 2026年5月29日

mogumap（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

収集する情報

本アプリは、外部サーバーへの個人情報の送信・収集を一切行いません。すべてのデータはお使いのデバイス内にのみ保存されます。

  • 位置情報: 場所を記録する際にデバイスの現在地を取得します。取得した位置情報はデバイス内にのみ保存され、外部に送信されることはありません。
  • 写真: 記録に添付した写真はデバイス内にのみ保存されます。
  • メモ・評価・その他の記録データ: すべてデバイス内にのみ保存されます。

ネットワーク通信

本アプリが行うネットワーク通信は以下の場合に限られます。

  • Google マップ URL の解決: ユーザーが短縮 URL を貼り付けた場合、リダイレクト先を解決するために HEAD リクエストを送信します。個人情報は含まれません。
  • Google マップアプリの起動: 地図表示時に Google マップアプリまたはブラウザを外部起動します。本アプリからデータの送信は行いません。

第三者への情報提供

本アプリは、ユーザーの情報を第三者に販売・提供・開示することはありません。

アナリティクス・広告

本アプリはアナリティクスツール・広告 SDK・クラッシュレポートツールを一切使用していません。

データの管理

アプリをアンインストールすることで、デバイス内のすべてのデータを削除できます。また、設定画面からデータを JSON 形式でエクスポートできます。
